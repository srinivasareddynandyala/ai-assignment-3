# Path Planning Algorithms

This repository contains basic Python implementations of pathfinding algorithms.

## Files
- `dijkstra.py` – Finds shortest path between cities using Dijkstra’s algorithm (CSV input)
- `ugv_static.py` – A* pathfinding on a grid with fixed obstacles
- `ugv_dynamic.py` – A* pathfinding with dynamic obstacles (replanning)

## How to Run
```bash
python dijkstra.py
python ugv_static.py
python ugv_dynamic.py
